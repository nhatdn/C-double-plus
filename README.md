4(1đ). Nhập điểm 3 môn cho sinh viên có mã sv nhập từ bàn phím.
5(0,5đ). Xuất ds sinh viên tăng dần theo ngày tháng năm sinh.

Lưu ý: Chuẩn hóa dữ liệu gồm: họ tên, ngày tháng năm sinh, mã sinh viên(phân biệt).
